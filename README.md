# Tier liste de Film avec AndroidStudio

Petit projet scolaire.
Le but est d'évaluer notre savoir-faire avec AndroidStudio.

L'application permet de parcours la base de données de IMDB et de gérer différentes Tier Liste de Film

#### Fonctionnalités :

		 Consulter les films ( Titre, Résumé, Image, Note )
		 Créer Tier Liste
		 Supprimer Tier Liste
		 Consulter une Tier Liste
		 Ajouter un film dans une Tier Liste
		 Supprimer un film d'une Tier Liste
